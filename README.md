### Landing-page-login

Projeto simples de landing page com sistema de login e cadastro para treinar o desenvolvimento web (front-end + back-end).

## Objetivo

Este projeto foi criado com foco em:

- Aprimorar habilidades de autenticação

- Praticar integração entre front-end e back-end

- evoluir no front-end para utilizar futuros novos frameworks e bibliotecas para o projeto

- Trabalhar com banco de dados

- Evoluir gradualmente para um sistema completo

---

## Tecnologias utilizadas

Front-end:

- HTML

- CSS

- TypeScript

Back-end:

- Node.js

- Express

- PostgreSQL

- prisma ORM

- TypeScript

---

## Futuras melhorias

- Autenticação com JWT

- Adição de frameworks e bibliotecas para o front-end

- Hash de senha com bcrypt

- Validação de dados com zod
